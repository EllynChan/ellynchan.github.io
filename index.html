Unity 3D VR Application
for Psychological Study

A 3D Unity application that integrates an experience with HTC Vive Cosmos using the SteamVR package. The users are tasked with walking to key destinations, memorizing key game objects, and using the laser pointer to interact with UI across 10 diverse scenes. Details of the study is published in the article "The Effects of Emotion on Retrospective Duration Memory using Virtual Reality."

2D Video Game

A 2D casual fishing collection game with an engaging storyline and turn-based battle mechanics, "Lord of the Lakes". The langauge used is C++ with GLFW and Dear ImGui libraries, using the Entity-Component-System design pattern. Awarded Honourable Mention by industry juries, and first place by Player's Choice votings.
